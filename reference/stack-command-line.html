Stack command line tool

Cloudspin stack command line tool reference

The cloudspin stack command line tool is included in the cloudspin-stack gem. It supports managing instances of a stack, using the stack project structure.

Installation

Install the gem, and the stack command should be available.

gem install cloudspin-stack
stack help

Commands

The most up to date documentation of commands and options is available by running stack help.

The tool assumes it is being run in a directory that follows the project structure as described in the stack project structure reference, and documentation files as in the stack instance configuration reference.
